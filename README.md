# 🛡️ Google Cybersecurity Certificate Labs: Incident Response Journal

Welcome to my lab projects! This repository serves as a practical portfolio documenting my hands-on experience throughout the **Google Cybersecurity Professional Certificate**. 

Instead of generic lab worksheets, each technical exercise here is structured as an **Operational Incident Response (IR) Journal**. This repository tracks my progression in:

* **Network Security:** Packet analysis and anomaly detection using Wireshark.
* **Security Automation:** Building defensive scripts and allow-list updaters with Python.
* **Data & Auditing:** Querying logs and managing security databases with SQL.
* **System Defense:** Hardening environments using Linux/Bash and analyzing access controls.
